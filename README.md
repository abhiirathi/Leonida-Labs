## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/abhiirathi/Leonida-Labs.git

# Navigate to directory
cd Leonida\ Labs

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
